# The Game aka pime aka alchemist

## Compilation configuration

Set variables in config.txt

## Build

```
./configure
make -j `nproc`
```

## Run all together

```
./start.sh
```

## Run

* run server

In server/build run: 

```
./server_pime
```

* SDL client

In SDL directory run:

```
./build/pime_SDL
```
